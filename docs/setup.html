Step-by-step

Setup sequence.

Do these once. The system handles the daily loop after that.

1

Download and open DailyOS

Install from GitHub Releases and launch once.

Gatekeeper: if blocked, run xattr -cr /Applications/DailyOS.app then reopen.

2

Connect Google

Use Settings to authorize Calendar and Gmail read access.

DailyOS does not send or delete messages without explicit action.

3

Choose workspace folder

Pick where your local operational memory will live.

~/Documents/DailyOS/
├── _today/
├── _inbox/
├── _archive/
├── Accounts/
└── Projects/
4

Add accounts and projects

Populate the entities you actually manage. Start small and expand gradually.

5

Open your first morning

Tomorrow's briefing will reflect today's context. By week two, preparation quality compounds.

Open the app. Your day is ready.
