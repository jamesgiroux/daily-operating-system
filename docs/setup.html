Step 6 of 13
6 Google API (Optional)

Connect Google services?

Calendar and Gmail integration powers the /today command.

What Google API enables

• See today's meetings in /today dashboard
• Scan emails for customer correspondence
• Create calendar events and time blocks

1
Open Google Cloud Console

Sign in with the Google account you use for calendar/email.

Open Console →
2
Create a New Project

Click project dropdown → "New Project" → name it "DailyOS" → Create

3
Enable the APIs

Go to "APIs & Services" → "Library" → Search and enable:
• Google Calendar API
• Gmail API
• Google Sheets API
• Google Docs API

Open API Library →
4
Configure OAuth Consent Screen

"APIs & Services" → "OAuth consent screen" → External → Create
Fill in app name, your email. Add yourself as test user.

Open OAuth Consent →
5
Create OAuth Credentials

"Credentials" → "Create Credentials" → "OAuth client ID"
Choose "Desktop app" → Create → Download JSON

Open Credentials →
6
Save Credentials File

Rename downloaded file to credentials.json
Move it to: .config/google/credentials.json
