# Figma Working Area

Use this directory for design-only artifacts created from or for Figma MCP work:

- export notes
- component and variable mapping tables
- screenshots used for design-system reconciliation
- temporary asset manifests
- audit notes for the in-Figma version of the DailyOS system

Do not put runtime app assets here. If the shipped app imports an asset, move it to `src/assets/` or `public/` according to how it is served and update the implementation.

Do not store customer-specific data, real domains, real company names, real email addresses, or production screenshots in this directory.
