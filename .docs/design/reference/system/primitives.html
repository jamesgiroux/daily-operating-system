Promotion Audit

The catalogue includes every primitive spec. Demos with promoted CSS render through scoped classes. The only remaining gap is left visible so a future promotion can replace the fallback without guessing.

  • 17 canonical-class demos
  • 1 promotion-needed primitive
  • InlineInput needs InlineInput.module.css
  • TypeBadge currently uses AccountHero source-local classes

A primitive should change here only when its canonical module changes.
